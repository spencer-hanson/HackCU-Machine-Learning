### Running the lab

To run this lab, simply execute the python script, in python2.7
~~~~
python Iris.py
~~~~


#### Packages needed for this lab
* scipy
* numpy
* matplotlib
~~~~
sudo pip install scipy numpy matplotlib
~~~~


#### The columns in this dataset are:
* Id
* SepalLengthCm
* SepalWidthCm
* PetalLengthCm
* PetalWidthCm
* Species
